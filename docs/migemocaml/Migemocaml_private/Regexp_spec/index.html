Module Migemocaml_private.Regexp_spec

Regexp_spec define module interface and some modules to generate regexp from Migemo module.

If you want to use Migemocaml in your application with Str module, you will need OCaml_str module. Create new module from S if you want to generate regexp for unsupported format in Migemocaml.
